# FoodWaste
Repositório dedicado ao projeto FoodWaste, com o objetivo análise e previsão de desperdício alimentar em restaurantes. Criado como ferramenta de aperfeiçoamento pessoal apresentando conhecimentos de ponta-a-ponta da área de dados, mas com potencial uso profissional em gestão de dados, automação e inteligência preditiva aplicada á restaurantes..
